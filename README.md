# Lab-Session
I created this example as part of a lab session demonstration for second-year students at my college. As a final-year student, I wanted to share some of the knowledge and experience I have gained during my studies and demonstrate practical Blueprint design concepts in Unreal Engine.The goal of this exercise is to demonstrate different approaches to communication between Blueprints and to introduce students to important software architecture concepts in Unreal Engine.

Learning Objectives

How to create simple gameplay mechanics using Blueprints.
How different actors in a level communicate with each other.
How to manage game state using a manager actor.
How to trigger events such as destroying actors and opening doors.
How to interact with actors.
How to apply damage and update values like health,score and gold.
How to use moddeling inside unreal for basic shapes. (Create basic shapes, Use boolean, use patern and merge actors)
How to Prototype a level with brushes.
UI for player overlay.

After implementing the basic version, the system will be refactored to demonstrate more advanced Unreal Engine techniques such as:

Inheritance ( polumorphic event, monomorphic event, Encapsulation).
Components.
Interfaces.
Dispatchers.

This exercise helps to understand how a simple system can evolve into a cleaner and more scalable architecture, which is an important concept when developing larger game systems.
